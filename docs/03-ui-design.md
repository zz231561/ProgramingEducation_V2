# UI/UX 設計

## 設計風格：GitHub / VS Code Dark

純 Dark Mode，參考 GitHub Dark 與 VS Code 的設計語言。
務實、熟悉、開發者友好，適合程式教學場景。

| 項目 | 規格 |
|------|------|
| 風格 | GitHub Dark + VS Code，搭配 shadcn/ui（dark preset） |
| 主題 | 純 Dark Mode，不提供 Light 切換 |
| 色彩 | Tailwind CSS 變數，GitHub Dark 色票 |
| 字型 | Inter (UI) + Noto Sans TC (中文) + JetBrains Mono (程式碼) |
| 元件庫 | shadcn/ui（基於 Radix UI，dark theme 客製） |
| 響應式 | 完整 mobile / tablet / desktop |
| 動畫 | 實用導向，CSS transition 為主 |

**導覽方式 — Top Navigation Bar（非 Sidebar）：**

採用 GitHub 風格的頂部 tab 導覽：
- 更節省水平空間（Workspace 頁面需要最大編輯區域）
- 頁面數量少（5 頁），tab 即可承載
- 與 GitHub / VS Code 的導覽體驗一致

**Tab 項目：**
- **Workspace** — 程式碼編輯 + AI 對話（預設頁）
- **Learn** — 結構化學習路徑
- **Quiz** — 智慧測驗
- **Knowledge** — 知識圖譜
- **Dashboard** — 學習總覽（教師可見班級管理）

Active tab: `border-bottom: 2px solid #F78166`（GitHub 橘色底線）

**關鍵互動：**
- Editor 與 Chat Panel 以 `react-resizable-panels` 拖曳調整寬度
- Output Panel 可收合為單行狀態列（顯示最後一行輸出）
- 點擊 `[▶ Run]` → Judge0 API → 結果顯示在 Output + 自動注入 AI Chat
- 點擊 `[🤖 AI]` toggle Chat Panel 顯示/隱藏
- Editor 使用 VS Code Dark+ 配色的 CodeMirror 6 主題（One Dark）
- Output tab 有未讀紅點提示（stderr 有新內容時）

## 響應式設計

| 斷點 | 寬度 | 佈局調整 |
|------|------|---------|
| Desktop | >= 1280px | 完整佈局（Editor + Chat side-by-side） |
| Laptop | 1024-1279px | Chat Panel 改為 overlay drawer（從右滑出） |
| Tablet | 768-1023px | 單欄佈局，Chat 為 bottom sheet |
| Mobile | < 768px | 全螢幕單欄，Header tab 改為 hamburger menu |
