## Page 1: Workspace（程式碼工作區）

```
┌──────────────────────────────────────────────────────────┐
│ Header                                                    │
├──────────────────────────────────────────────────────────┤
│ Toolbar: [Files ▾] [C++ ▾]        [stdin] [▶ Run] [AI] │
├─────────────────────────────┬────────────────────────────┤
│  Editor (CodeMirror 6)      │  AI Chat Panel (400px)     │
│  bg: #0D1117                │  bg: #161B22, 可收合       │
│  VS Code Dark+ (One Dark)   │                            │
│                              │  Coddy / 學生 對話氣泡  │
│                              │                            │
├─────────────────────────────┤  輸入框 + Send 按鈕        │
│  Output (bg: #010409, 可收合)│                            │
│  [stdout|stderr|compile] tab │                            │
└─────────────────────────────┴────────────────────────────┘
```

- Editor / Chat 以 `react-resizable-panels` 拖曳調整
- Output 可收合為單行狀態列
- [▶ Run] → Judge0 → 結果顯示在 Output + 自動注入 Chat
- stderr 有新內容時 tab 顯示紅點
